# 🔬 Cell Organelle Explorer

An interactive AI-powered biology app that lets users explore cell organelles using hand gestures and camera input.

## 🚀 Features

- ✋ Hand gesture recognition (MediaPipe)
- 🧠 Interactive quiz system
- 🎥 Live camera-based interaction
- 📱 Installable as a Progressive Web App (PWA)
- ⚡ Offline support (via Service Worker)

---

## 📦 How to Use

### 1. Clone the repository

git clone https://github.com/your-username/organelle-explorer.git

### 2. Run locally

npx serve

OR

python -m http.server

Then open:

http://localhost:3000

---

## 📲 Install as App

1. Open deployed link (GitHub Pages / Netlify)
2. Click **Add to Home Screen**
3. App works like a native app
